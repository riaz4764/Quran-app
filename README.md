# 📖 Quran Al-Kareem Website

An elegant, fast, and fully responsive offline-capable Holy Quran web application built for browsers. The website features a beautiful modern dark UI, comprehensive Surah indexing, real-time searching, and smooth multi-tab navigation.

🚀 **Live Link:** [https://riaz4764.github.io/Quran-app/](https://riaz4764.github.io/Quran-app/)

---

## ✨ Features

* **Complete Quran Index:** Full list of 114 Surahs with detailed metadata (Ayahs count, Revelation type - Meccan/Medinan).
* **Instant Search:** Real-time search functionality to quickly find Surahs by name or number.
* **Modern Dark UI:** Premium aesthetic design optimized for browser readability and comfort during night reading.
* **Multi-Tab Navigation:** Easy switching between Quran, Search, Bookmarks, Qibla, and Settings.
* **GitHub Pages Optimized:** Fully configured to deploy and load seamlessly on any device browser.
